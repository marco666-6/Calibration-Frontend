# Calibration System Frontend

Vite React frontend for the Calibration System.

## Main Areas

- Equipment master data
- Internal and external calibration plans
- Actual calibration result recording
- Three-step approval workflow
- Locking and PDF download
- Sections, external companies, users, and calibration roles

## Local Development

```bash
npm install
npm run dev
```

The API endpoint is configured through `VITE_API_ENDPOINT`.
